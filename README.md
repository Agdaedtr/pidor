# 🐱 Кошачье 2048 — Android APK

Мобильная игра-головоломка про котиков, упакованная в Android-приложение.
Игровая логика — один HTML5-файл (`www/index.html`), обёрнутый в нативный WebView через **Capacitor**.
APK собирается автоматически в **GitHub Actions** (см. вкладку *Actions*).

## 📥 Где взять готовый APK

После того как пройдёт сборка (вкладка **Actions** → последний запуск **Build Android APK**):

- **Артефакт сборки:** внизу страницы запуска, блок *Artifacts* → `koshki-2048-apk`.
- **Релиз:** вкладка **Releases** → `Кошачье 2048 v1.0.0` → файл `koshki-2048.apk`.

## 📲 Установка на телефон

1. Скачай `koshki-2048.apk` на Android-телефон.
2. Открой файл — система предложит разрешить установку из этого источника (*Настройки → Разрешить установку неизвестных приложений*).
3. Установи и играй. 🐾

> Это debug-сборка (подписана отладочным ключом) — её достаточно для установки и игры, но не для публикации в Google Play.

## 🎮 Как играть

Свайпай — одинаковые котики сливаются и эволюционируют: 🐱 → 😺 → 😻 → … → 🦁 Лев (2048) → 🐯 → 👑. Цель — дорастить до Льва и побить рекорд.

## 🛠️ Собрать самому

```bash
npm install
npx cap add android
npx cap sync android
cd android && ./gradlew assembleDebug
# APK: android/app/build/outputs/apk/debug/app-debug.apk
```
Нужны: Node.js, JDK 17, Android SDK (platform 34, build-tools 34).
